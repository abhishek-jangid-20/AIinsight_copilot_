# Phase 2: MiniGPT Transformer Lab

This module is intentionally separate from the production repository assistant.

The production assistant uses external LLM APIs plus embeddings and retrieval. The MiniGPT lab exists for education and research: tokenizer, vocabulary builder, token and positional embeddings, causal self-attention, multi-head attention, feed-forward blocks, layer normalization, training loop, generation, loss charts, and attention visualization.

Do not wire MiniGPT into the main RAG response path.
